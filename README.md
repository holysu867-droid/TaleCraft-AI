# 童话之声 (FairyTaleVoice) - API 自动化测试仓库

**所属团队：** 第15组
**测试负责人：** 杨嘉仪 (QA)

## 📌 仓库说明
本模块由测试工程师(QA)负责，主要用于 Sprint 1 迭代中的接口自动化测试与质量保障。不包含前后端业务代码，专注保障系统集成质量。

## 🧪 测试策略说明
由于当前阶段后端业务代码仍在开发集成中，本测试工程采用了**契约驱动测试（Contract-Driven Testing）**策略。
所有测试脚本均基于团队共同敲定的《API接口文档》编写。我们在后端联调前提前准备好了测试用例与断言代码（实现了**测试左移**）。

## ⚙️ 涵盖的测试范围
- **`POST /api/story/ai`**：校验必填字段拦截、基于已有章节续写的逻辑断言。
- **`POST /api/audio`**：针对 `speechRate`（语速）字段执行了等价类和**边界值测试**（覆盖0.5~2的有效区间与越界情况）。
- **安全鉴权测试**：无 Token 及无效 Token 的访问拦截验证。

## 🚀 测试环境与运行方式
- Python 3.12+
- 测试框架: `pytest`
- HTTP客户端: `requests`

1. 安装依赖包：
```bash
pip install -r requirements.txt
```

2. 运行所有用例并生成 HTML 测试报告：
```bash
pytest tests/ -v
```
